4. 이력서/자기소개서 최종 수정 (1시간)

이력서 수정:
□ [이메일 주소] → 실제 이메일 입력
□ [전화번호] → 실제 번호 입력
□ 내용 검토 및 수정
□ PDF로 저장 (Cmd + P → PDF 저장)

자기소개서 수정:
□ [회사의 대표 게임/특징] → 지원 회사에 맞게 수정
□ [회사의 강점/비전] → 지원 회사에 맞게 수정
□ [이메일 주소] → 실제 이메일 입력
□ [전화번호] → 실제 번호 입력
□ 전체 내용 검토
□ PDF로 저장

파일 위치:
• 이력서_여지민.html
• 자기소개서_여지민.html
